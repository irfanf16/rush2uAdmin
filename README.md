# Rush2U Admin Dashboard

Operations admin panel for the **Rush2U** on-demand delivery platform. Monitors deliveries, manages drivers, controls notifications, and tracks earnings.

![Laravel](https://img.shields.io/badge/Laravel-FF2D20?style=flat&logo=laravel)
![Bootstrap](https://img.shields.io/badge/Bootstrap-7952B3?style=flat&logo=bootstrap)
![MySQL](https://img.shields.io/badge/MySQL-4479A1?style=flat&logo=mysql)

## Features

- Real-time delivery tracking map
- Driver management and assignment
- Operations dashboard and analytics
- Push notification management
- Earnings and payout control

## Getting Started

```bash
composer install
cp .env.example .env && php artisan key:generate
php artisan migrate && php artisan serve
```
